Дока по команде ls

ls - список файлов в текущем каталоге.
ls -l список файлов в текущем каталоге с полной формой вывода
ls -la список файлов в текущем каталоге с полной формой вывода и выводом скрытых файлов
ls -R рекурсивный вывод списка файлов в текущем каталоге
ls -ld */ вывод всех каталогов в текущей папке
ls -lt вывод файлов в текущем каталоге с сортировкой по времени модификации
ls -ltr вывод файлов в текущем каталоге с сортировкой по времени модификации в обратном порядке.
но в скриптах правильнее сортировать так: --sort=extension | --sort=size | --sort=time | --sort=version - будет понятнее читать.
ls -l --author выводит пользователя создавшего файл.
ls -ltc показывает время модификации файла.
ls -lh вывод размера файла в виде 2K, 20M 1,4G - удобным для восприятия.

--------------------------------------------------------------------------------------------------------------------------------
В полном выводе первое значение в группе с правами может быть:

- - Regular file
b - Block special file
c - Character special file
d - Directory
l - Symbolic link
n - Network file
p - FIFO
s - Socket

И последнее значение:

s - setgid bit
t - sticky bit
+ - ACL
